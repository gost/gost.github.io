More Info

OGC SensorThings API

An Open Geospatial Consortium (OGC) standard providing an open and unified framework to interconnect IoT sensing devices, data, and applications over the Web. It is an open standard addressing the syntactic interoperability and semantic interoperability of the Internet of Things. GOST is an OGC certified implementation on all levels. See OGC GOST Product details
